WordsRemem Website
This is an interesting website which helps Chinese students learning English new words.
The database was built by Microsoft SQL Server 2005. The ASP.NET website was programmed in C# language.
Features

The Login/PasswordsRetrieve/Registration interface :
1. You may create you account first by click the 注册 button.
2. You can log in and start your learning.
3. You can get back your passwords by click the 忘记密码 button if you forget it.



The MainPage :
You can choose one database of English words to start your learning. The databases contains types of popular exams such as TOEFL and GRE.



The Function interface(quiz interface) :
There are three buttons you can click on the page. 我认识 means you have ever learnt this word and you want to skip it. 没见过 means this is a totally unfamiliar word to you. 确定 means OK and will pass your answer back.
